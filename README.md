# FormBridge AI 🚀

FormBridge AI is an advanced AI platform designed to make document handling effortless. From AI-assisted form filling to CV building and document explanation, FormBridge AI bridges the gap between static documents and intelligent automation.

---

## ✨ Features

- 📄 **AI Form Filling**: Intelligent data extraction and field population for PDFs and images.
- 📝 **AI CV Builder**: Create world-class resumes with the help of GPT-4.
- 💡 **Document Explainer**: Understand complex contracts and documents in seconds.
- 🌍 **Multilingual**: Native support for English, German, and Arabic.
- 🤳 **Mobile First**: Fully operational on Web, Android, and iOS.

---

## 🛠️ Quick Start

### 1. Installation

```bash
npm install
```

### 2. Environment Setup

Configure your `.env.local` using `.env.example`:
- `OPENAI_API_KEY`
- `STRIPE_SECRET_KEY`
- `JWT_SECRET`

### 3. Run Development

```bash
npm run dev
```
